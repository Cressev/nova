## 放置规则

- 只有用户明确要求写某份预研文档时，Agent 才能写入正文。

## 维护方式

- 新增预研文档时，文件名使用实际创建日期，格式为 `YYYYMMDD_主题.md`。
- 新增文件后，更新本 README 的“当前文件”。
- 不主动补写普通文档内容。
