# references 目录说明

## 用途

本目录保存项目记忆持久化工作流的参考模板和操作说明。

## 当前文件

- `agents_md_template.md`：AGENTS.md 初始化结构说明。
- `bootstrap_prompt_template.md`：新项目初始化提示词模板。
- `todolist_workflow_template.md`：TODOList 工作流说明。
- `user_queries_workflow_template.md`：用户请求入口工作流说明。
- `log_workflow_template.md`：追加式日志工作流说明。

## 放置规则

- 只放长期可复用的流程模板和参考说明。
- 不放项目临时记录。
- 不放用户任务原文，用户任务原文只写入 `user-queries.md`。
