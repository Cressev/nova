# 项目记忆持久化初始化提示词模板

当需要在新项目中初始化项目记忆持久化结构时，可以使用以下提示：

```text
请按 project-memory-persistence 工作流为当前项目初始化持久记忆系统。

要求：
1. 检查现有项目结构。
2. 创建或维护根目录核心资产：AGENTS.md、CURRENT.md、PROGRESS.md、log.md、TODOList.md、user-queries.md。
3. 创建 findings/、reports/、references/，并为 durable 目录添加 README.md。
4. 按模板填写 AGENTS.md，包括项目摘要、路径、任务入口、TODOList、日志、报告、git 策略、文件安全、用户偏好、风险边界和待确认问题。
5. 所有文档正文使用中文。
6. 初始化后记录必须向用户确认的问题，不要伪造答案。
```
